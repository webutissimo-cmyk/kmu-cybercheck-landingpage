# kmu-cybercheck-landingpage
Landingpage für die Domain kmu-cybercheck.de als schlanke Vorverkaufs- und Weiterleitungsseite zum KMU-SicherheitsRadar. Ziel ist es, Geschäftsführer, Inhaber und KMU-Verantwortliche für einen kostenlosen Cyber-Sicherheitscheck zu gewinnen und auf https://cyber-sicherheitsradar.de/check weiterzuleiten.
